=== Demonstration of Example

  ✓ `EntityManager.createQuery(criteriaQuery)`::
    . Find native query
    . Convert `List<Object[]>` to desired object

  ✓ `EntityManager.createNativeQuery(sqlString, resultClass)`::
    . Run insert statement
    . Run select statement
    . Convert `jakarta.persistence.Tuple.java` to desired object
